# Arrows - Puzzle Escape (Web)

A puzzle game inspired by arrow-maze escape mechanics.

## Gameplay

- You start at `S` (top-left) and must reach `E` (bottom-right).
- Each tile has a direction arrow.
- Click tiles to rotate arrows clockwise.
- Press `Trace Escape` to run the route simulation.
- Avoid dead-ends and loops.

## Features

- Procedurally generated levels
- Increasing grid size with level progression
- Hint system (auto-fixes one tile)
- Trace animation and failure reasons
- Polished responsive UI (desktop + mobile web)

## Run

```bash
yarn
yarn dev
```

## Build

```bash
yarn build
```
